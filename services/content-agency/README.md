## 🎯 Quick Start (30 Minutes)

### Step 1: Deploy Landing Page
```bash
# Copy landing page to your domain
cp landing-page.html /var/www/html/index.html

# OR deploy to Cloudflare Pages:
# 1. Create account at dash.cloudflare.com
# 2. Upload landing-page.html
# 3. Get free domain: kisii-ai-services.pages.dev
```

### Step 2: Setup WhatsApp Bot
```bash
# Run the setup script
chmod +x setup-whatsapp-bot.sh
./setup-whatsapp-bot.sh

# Start the bot
cd ~/whatsapp-bot
./start.sh

# Scan QR code with your WhatsApp
# Bot is now live 24/7!
```

### Step 3: Setup Content Automation
```bash
# Run content setup
chmod +x setup-content-automation.sh
./setup-content-automation.sh

# Generate content for a client
./generate-daily-content.sh "Client Name" "Industry"
```

### Step 4: Start Outreach
- Open `outreach-scripts.md`
- Copy Template 1 (First Contact)
- Send to 5 local businesses today
- Follow up tomorrow

## 🤖 How the Automation Works

### WhatsApp Bot Flow:
1. Customer messages you

3. Bot sends appropriate response instantly
4. Complex questions → Bot notifies you
5. You take over when needed

### Content Generation Flow:
1. Run `./generate-daily-content.sh "Client" "Industry"`
2. Script creates 3 posts for the day
3. Customize with client details

## 🚨 Troubleshooting

### WhatsApp Bot Issues:
**Problem:** QR code not showing
**Solution:** Install dependencies: `npm install`

**Problem:** Bot not responding
**Solution:** Check logs: `pm2 logs kisii-whatsapp-bot`

### Content Generation Issues:
**Problem:** Script not running
**Solution:** Make executable: `chmod +x setup-content-automation.sh`

### Landing Page Issues:
**Problem:** Not displaying correctly
**Solution:** Check file path and open in browser
